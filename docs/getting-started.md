# Installation {#installation}

Start a new project:

```bash
$ mkdir ladder-demo && cd ladder-demo
```

Initialize your project:

```bash
$ npm init
```

Add the LadderJS dependency:

```bash
$ npm install --save ladderjs
```

Add the LadderJS command line shortcut in the`package.json`:

```js
{
    ...
    "scripts": {
        ...
        "ladderjs": "ladderjs"
    },
    ...
}
```
